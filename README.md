# Number Puzzle Game with Scoreboard Saved to File
## Introduction
- The Number Puzzle Game is a simple console-based mini project developed in C. The 
player guesses a randomly generated number, and the number of attempts is recorded. The 
final score is saved permanently using file handling.
## Objectives
- To create a number guessing game in C
- To use random number generation
- To store and display scores using files
- To understand basic C programming concepts
## System Features
- Random number generation
- Player name input
- Guess validation with hints
- Attempt counting
- Score saving and scoreboard display
- Menu-driven program
## Concepts & Technologies Used
- C programming language
- Functions, loops, and conditionals
- Arrays and strings
- File handling
- Rand() and srand()
## System Design
- playGame(): Handles the guessing logic
- saveScore(): Saves name and attempts to file
- showScoreboard(): Displays saved scores
- main(): Controls menu and program flow
## Advantages
- Simple and user-friendly
- Permanent score storage
- Improves logical thinking
- Useful for beginners
## Future Enhancements
- Difficulty levels
- Sorted scoreboard
- Date and time display
## Conclusion
- This project demonstrates the use of random numbers, file handling, and structured 
programming in C. It is an effective learning project for beginners and can be enhanced 
with additional features.
